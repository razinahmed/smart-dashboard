# Smart Dashboard

A real-time analytics dashboard built with React, Chart.js & Node.js — live weather, stocks, and news in one sleek UI.

## Features

- Live weather data with location detection
- Real-time stock market tracker
- News headlines aggregator
- Dark/Light theme toggle
- Responsive design for all devices
- Interactive charts and graphs

## Tech Stack

- **Frontend:** React, Chart.js, Tailwind CSS
- **Backend:** Node.js, Express
- **APIs:** OpenWeatherMap, Alpha Vantage, NewsAPI
- **Deployment:** Docker, Vercel

## Getting Started

```bash
git clone https://github.com/razinahmed/smart-dashboard.git
cd smart-dashboard
npm install
npm run dev
```
